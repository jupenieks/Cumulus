Account Related Classes

this is some general help information about the classes related to Accounts in the NPSP.

  • ACCT_Accounts_TDTM - Trigger Handler
  • ACCT_HH_Accounts_TEST - Test code for Household Accounts
  • ACCT_IndividualAccounts_TDTM - Trigger Handler that manages creating the appropriate Account for Contacts, when they are created or updated. This handles all three Account Models (Individual Bucket, One-to-One, and Household Account).
  • ACCT_ViewOverride_CTRL - Account action override for the View command, to allow the NPSP to redirect url's to the One-to-One account back to the Contact.
  • ACCT_ViewOverride_TEST = Test code for the Account view override.
